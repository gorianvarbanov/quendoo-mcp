## Key Points

1. **Quendoo API Key Entry:**
   - OLD: Every 24 hours
   - NEW: Once during registration

2. **JWT Token Entry:**
   - Never expires (30-day validity, but auto-renewed)
   - Entered once in Claude Desktop config

3. **Total Credentials:**
   - OAuth login (email/password) â†’ Proves identity
   - Quendoo API key â†’ Entered ONCE during registration
   - NOT entered twice!
